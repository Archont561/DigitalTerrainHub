ODM Options Cheat Sheet

🏗️ Modeling & Reconstruction

  • 3d-tiles – Generate OGC 3D Tiles outputs.
  • skip-3dmodel – Skip full 3D model generation.
  • fast-orthophoto – Generate orthophoto directly from sparse reconstruction.
  • mesh-octree-depth – Control mesh octree depth.
  • mesh-size – Set maximum mesh vertex count.
  • texturing-keep-unseen-faces – Keep mesh faces not visible in any image.
  • texturing-single-material – Use single material for OBJs.
  • texturing-skip-global-seam-leveling – Disable seam leveling.
  • use-3dmesh – Use full 3D mesh for orthophoto.
  • align <path> – Align output to existing DEM/LAS/LAZ file.

📸 Camera & Image Handling

  • camera-lens, cameras, feature-quality, feature-type
  • min-num-features, force-gps, geo, gcp
  • primary-band, use-exif, use-fixed-camera-params
  • use-hybrid-bundle-adjustment, rolling-shutter, rolling-shutter-readout

🧠 AI-Based Enhancements & Filtering

  • bg-removal – Remove background using AI.
  • sky-removal – Remove sky using AI.
  • auto-boundary – Auto-set boundary based on shot locations.
  • auto-boundary-distance – Set distance for auto-boundary.

🗺️ Geospatial Output & DEM

  • dem-resolution, dem-decimation, dem-gapfill-steps
  • dem-euclidean-map, dsm, dtm
  • boundary, smrf-scalar, smrf-slope, smrf-threshold, smrf-window

🧮 Point Cloud Processing

  • pc-classify, pc-copc, pc-csv, pc-ept, pc-las
  • pc-filter, pc-sample, pc-rectify, pc-quality
  • pc-skip-geometric

🖼️ Orthophoto Options

  • orthophoto-compression, orthophoto-resolution, orthophoto-cutline
  • orthophoto-kmz, orthophoto-no-tiled, orthophoto-png
  • skip-orthophoto

📁 Project & File Handling

  • project-path, name, copy-to
  • split-image-groups, help, version

⚙️ Performance & Settings

  • max-concurrency, optimize-disk-space, ignore-gsd
  • matcher-neighbors, matcher-order, matcher-type
  • sfm-algorithm, sfm-no-partial

🔄 Pipeline Control

  • end-with, rerun, rerun-from, rerun-all

🔀 Split & Merge

  • split, split-overlap, merge, sm-cluster, sm-no-align

📽️ Video Support

  • video-limit, video-resolution

🌞 Radiometric Calibration

  • radiometric-calibration, skip-band-alignment
